mc_xclip
========

Syncs clipboards of Midnight Commander and X Window System.

Install:

    sudo apt-get install xclip
    sudo pip install mc_xclip

    # If you use Display Manager:
    echo 'mc_xclip &' >> .xprofile

    # Else:
    echo 'mc_xclip &' >> .xinitrc

    # Reboot.
